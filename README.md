 📚 RAG-BOT | AI-Powered PDF Chatbot

A smart chatbot that lets you upload any PDF and ask questions from it in seconds. 
Built for students, researchers, and anyone who works with long documents.

 What it does
- Upload PDFs: Add any research paper, book, or notes
- Ask Anything: Type questions in plain English and get instant answers 
- Smart Search: Uses AI to find the exact information from your documents
- Fast & Clean UI: Simple Streamlit interface, no clutter

 Tech Stack
Frontend: Streamlit  
AI/LLM: LangChain + Groq API  
Vector DB: FAISS for fast document search  
Embeddings: HuggingFace sentence-transformers

 How to Run This Project
1. Clone the repository
2. Install requirements: `pip install -r requirements.txt`
3. Add your Groq API key in `.env` file 
4. Run the app: `streamlit run app.py`

 Project Structure
`app.py` - Main Streamlit app  
`requirements.txt` - All Python dependencies
